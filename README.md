<h2>This Is My Solo Project,This Project Was Completed With-In 10 Days</h2>

<ol>

<li>Mongodb</li>
<li>Express.js</li>
</ol>

<h2>Tech Stack Frontend</h2>
<ol>
<li>React.js</li>
<li>JavaScript</li>
<li>Express.js</li>
<li>Context Api</li>
<li>Tailwind CSS</li>
</ol>
